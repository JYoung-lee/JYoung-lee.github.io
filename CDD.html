프로젝트명 : 캣댕댕 ( 반려동물 SNS )


프로젝트 일정 : 2021.03.08 ~ 2021.04.15

개발목표

  • 나의 반려동물을 소개하고 반려 동물의 지식을 얻고싶은 사람들에게 공유
  • 상황이 되지 않아 키우지 못하는 이용자에게는 대리만족
  • 키우기 전 일지를 통하여 여러 가지의 지식을 배울 수 있다.
  • SNS 형식의 웹사이트로 누구나 쉽게 접근할 수 있고 정보를 얻을 수 있게 한다.

개발환경

  • OS : Window 10
  • Language/Skills : Java, JSP, jQuery & Ajax,HTML,CSS
  • Framework : Spring Framework , Bootstrap Framework
  • DBMS : Oracls.11g
  • Server : Tomcat8.5
  • 개발 Tool : Eclipse, Dbeaver

구현기능

  • 회원 : 로그인, 회원가입, 회원수정, 북마크,Feed,AllFeed,내 프로필
  • 관리자 : 신고 접수, 회원 관리, 반려동물추가

담당역할

  • 내 프로필 : 내 프로필 화면 구상 /펫 정보 CRUD/ 일지 CRUD / 일정 CRUD /

프로젝트 코드

  • GitHub : https://github.com/JYoung-lee/CDD

화면 및 구현기능 설명

  • 내 프로필 화면 - 달력의 날짜 클릭시 해당 일지와 일정을 볼 수 있다.
  • 달력API 사용, 클릭이벤트 자바스크립트 처리

화면 및 구현기능 설명

  • 일지 화면 - 펫의 하루일지 등록
  • 부트스트랩 Modal 사용, 날짜 선택 : datepicker 사용

화면 및 구현기능 설명

  • 일지 리스트 화면 - 유저의 모든 일지 목록 리스트 수정 / 삭제
  • 부트스트랩 Modal 사용,수정은 등록창을 수정창으로 사용,삭제는 Ajax 비동기 처리하여 삭제

화면 및 구현기능 설명

  • 일정 등록 화면 - 펫의 일정 등록
  • 부트스트랩 Modal 사용, 달력 API 사용하여 날짜 클릭이벤트 가능하도록 구현

화면 및 구현기능 설명
