지치지 않는 개발자 이준용 입니다.

프로젝트명 : Helimy


프로젝트 일정 : 2021.01.07 ~ 2021.02.05

개발목표

  • 코로나로 인하여 큰 타격을 받는 헬스장의 이익창출
  • 인원 몰림 현상 방지로 바이러스 전염방지효과↑
  • 주변 근처 예약 시스템을 통하여 사용자, 수요자를 만족할 수 있 는 신뢰성 있는 사이트 구축

개발환경

  • OS : Window 10
  • Language/Skills : Java, Jsp, Javascript, SQL, HTML, CSS
  • DBMS : Oracls.11g
  • Server : Tomcat8.5
  • 개발 Tool : Eclipse, Dbeaver

구현기능

  • 로그인, 회원가입, 회원수정, 즐겨찾기,예약내역 게시판, 결제내역
  • 매장 등록, 매장관리, 회원관리, 리뷰 관리,예약 관리, 결제 관리

담당역할

  • 매장등록 / 관리, 결제, 예약 등록/취소

프로젝트 코드

  • GitHub : https://github.com/JYoung-lee/Helimy

화면 및 구현기능 설명

  • 메인 화면 - 주변 헬스장을 검색할 수 있는 검색 기능

화면 및 구현기능 설명

  • 매장 메인 화면 - 해당 헬스장 정보, 구매, 예약 기능

화면 및 구현기능 설명

  • 상품 결제 화면 - 해당 헬스장 회원권 구매 기능

화면 및 구현기능 설명

  • 예약 화면 - 원하는 시간 예약 가능 (CheckBox 3개 이상X , 3시간 이상X 스크립트 처리)

화면 및 구현기능 설명

  • 매장등록 화면 - 매장 대표이미지, 매장정보 등록

화면 및 구현기능 설명

  • 매장 시설등록 화면 - 홍보글,공지글,편의시설,부가서비스 등록

화면 및 구현기능 설명

  • 매장 이미지등록 화면 - 사진으로 홍보할수 있도록 등록 가능
